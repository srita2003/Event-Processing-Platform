# 🚀 Event_Processing_And_Monitoring_Dashboard

## 📌 Overview
This project is a microservices-based event processing system using:
- Node.js
- Docker
- RabbitMQ
- MongoDB
- Prometheus & Grafana

## 🔧 Features
- Send events from frontend
- Queue processing using RabbitMQ
- Store events in MongoDB
- Analytics dashboard
- Monitoring using Prometheus & Grafana

## 🏗️ Architecture
Frontend → API Gateway → Event Service → RabbitMQ → Worker → MongoDB → Analytics

## 🚀 Run Project
docker-compose up --build

## 🌐 Services
- API Gateway: http://localhost:8080
- RabbitMQ UI: http://localhost:15672
- Grafana: http://localhost:3000
- Prometheus: http://localhost:9090
